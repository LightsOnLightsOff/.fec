## Available Scripts
```sh
npm install
```


Duplicate `config-example.js` in the root directory\
Rename to `config.js`\
Replace the `TOKEN_HERE` to your API Token and save


To create a production build:

```sh
npm run build
```


To create development build:

```sh
npm run server
```
